# ここにいるワン MVP

仲の良いフレンドと一緒にドッグランへ！予定を立てて、みんなで行こう。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## デモ用ログイン

ログインページ（`/login`）で3人のデモユーザーから選択してください：

| ユーザー | 犬 | 友達コード |
|---------|---|-----------|
| はなこ | モカ（トイプードル） | WOOF-1234 |
| けんた | レオ（ゴールデンレトリバー） | BARK-5678 |
| さくら | はな（柴犬） | PAW-9012 |

初期状態では、全員がフレンド関係にあり、今日の予定がいくつか入っています。

## ルーティング

| パス | 説明 |
|-----|------|
| `/` | ホーム（フレンド予定 + 近くのドッグラン） |
| `/parks/[parkId]` | 施設詳細 |
| `/plans/new?parkId=...` | 予定作成 |
| `/plans/[planId]` | 予定詳細（参加・チャット） |
| `/friends` | フレンド管理 |
| `/notifications` | 通知一覧 |
| `/me` | マイページ |
| `/me/edit` | プロフィール編集 |
| `/login` | ログイン（デモ） |

## 技術スタック

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **localStorage** – バックエンドなし、全データはブラウザに保存

## 主な機能

- 🐾 **フレンド予定カード** – 横スクロール、迷い中は黄色強調
- 📍 **施設詳細** – 今日の予定（確定/迷い中/匿名）カウント
- ✅ **予定作成** – 日付・時間帯・状態・一言
- 💬 **チャット** – 参加・確定で自動システムメッセージ
- 👥 **フレンド** – 友達コード入力のみ（検索なし）
- 🔔 **通知** – 参加・確定・フレンド申請
- 📱 **モバイルファースト** – 最大幅 `max-w-lg`
